# 1. 서비스 개요

## 서비스명
Campus Notice AI

## 서비스 설명

대학생이 학교 홈페이지 학사공지, 장학금, 수강신청, 비교과 프로그램 등의 공지를 놓치지 않도록 대학 홈페이지 공지를 자동 수집하고, 검색·북마크·키워드 구독 기능을 제공하는 AI 기반 통합 학사공지 플랫폼이다.

---

# 2. 5W1H 분석

| 항목 | 내용 |
|---|---|
| Why | 대학 공지는 여러 게시판에 분산되어 있으며 갱신 주기가 빨라 놓치기 쉽다. 기존 방식(학교 홈페이지 직접 확인, 커뮤니티 공유)은 실시간 반영이 어렵고 개인 관심사 맞춤 기능이 부족하다. |
| Who | 대학생 전체, 공지 의존도가 높은 재학생 |
| What | 공지 통합 조회, 상세 조회, 검색, 북마크, 키워드 구독 |
| How | 크롤러 → DB 저장 → 프론트 출력 → 구독/스크랩 제공 |
| Stack | Frontend / Backend / AI / Infra 분리 |
| When | MVP 4주 |

# 3. 사용자 정의 (Who)

## 핵심 타겟

- 대학생
- 학사공지 의존도가 높은 재학생
- 장학금·비교과 활동 참여 학생

---

## Persona A

이름: 김민수 (22세 / 컴퓨터공학과)

목표:
- 장학금 공지 놓치지 않기
- 비교과 프로그램 빠르게 확인

Pain Point:
- 홈페이지를 매일 방문하기 어려움
- 공지가 여러 게시판으로 분산됨

---

## Persona B

이름: 이지은 (24세 / 졸업예정)

목표:
- 졸업요건 및 수강신청 일정 확인
- 취업 프로그램 추적

Pain Point:
- 과거 공지 재검색 어려움
- 중요 일정 누락
